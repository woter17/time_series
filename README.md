# Временные ряды: лабораторные работы

Репозиторий содержит 12 лабораторных работ по дисциплине «Временные ряды». Материалы оформлены в единой структуре `lab_01` ... `lab_12` и рассчитаны на запуск в Jupyter Notebook или Google Colab.

## Содержание

| Папка | Тема |
|---|---|
| `lab_01` | Генерация шумов и сигналов |
| `lab_02` | Первичный анализ, тренд, сезонность и автокорреляция |
| `lab_03` | Стационарность, Box-Cox и авторегрессия |
| `lab_04` | Скользящее среднее, ARMA/ARIMA, пропуски и выбросы |
| `lab_05` | Лаговые признаки, Random Forest, naive baseline и аномалии |
| `lab_06` | DTW и классификация временных рядов |
| `lab_07` | SAX/SFA, BOSS/WEASEL, ROCKET, cBOSS, 1D-CNN |
| `lab_08` | Shapelet Transform |
| `lab_09` | Спектральные и кепстральные аудиопризнаки |
| `lab_10` | Фильтрация сигнала и экспоненциальное сглаживание |
| `lab_11` | Подготовка данных для LSTM |
| `lab_12` | VAR и многомерные временные ряды |

## Запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
jupyter notebook
```

Часть работ требует тяжёлых библиотек: `torch`, `sktime`, `pyts`, `dtaidistance`, `sounddevice`, `librosa`. Если они не нужны для конкретной проверки, можно запускать только соответствующие ноутбуки.
